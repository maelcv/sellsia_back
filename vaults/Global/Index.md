# Global Folder
Ce dossier (dossier /Global/) contiens toutes les données globales accesibles aux differents agents IA de la plateforme BoatSwain.
Cet index decris l'utilité et le contenu de chacun des sous dossier de Global.

# Index
## Users 
Ce dossier contiens les sous dossiers des préférences utilisateur. Chaque dossier représente un utilisateur et est nommé par l'id de l'utilisateur qu'il represente. Dans chaque dossier se trouve les préférences de de l'utilisateur pour ses conversations avec les agents et les données jugées pertinentes que les agents ont recueillis sur ce dernier.

## Prompt
Ce dossier est protéger, dedans se trouvent les prompts system des agents ia (prompt injecté avant les consignes personnalisé et les messages des utilisateurs). Aucun utilisateur ou agent ia ne peut éditer le contenu de se dossier ou des fichiers qui s'y trouvent.
